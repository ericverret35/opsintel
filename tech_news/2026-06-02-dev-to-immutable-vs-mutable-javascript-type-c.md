## Immutable vs Mutable ||JavaScript Type Conversion

**Source**: Dev.to

  Immutable vs Mutable (Non-Immutable)  

  Immutable  

 
 An immutable value cannot be changed after it is created. 
 When you modify it, JavaScript creates a new value instead of changing the original one. 
 

  Example  
 let a = "Hello"; 
a = "Hi";  
What happens? 
 "Hello"   // old value remains unchanged 
"Hi"      // new value is created  
The variable a now points to the new value "Hi". 

  Immutable Data Types  

 
 String 
 Number 
 Boolean 
 Null 
 Undefined 
 Symbol 
 BigInt
