<p><strong>THIS IS A SCHEDULED EVENT Jun <var>22</var>, <var>12:00</var> - <var>13:00</var> UTC</strong></p> <p> <small>Jun <var>10</var>, <var>20:16</var> UTC</small><br /> <strong>Scheduled</strong> - On 2026-06-22 from 12:00-13:00 UTC, Cloudflare will be performing scheduled maintenance on the data store responsible for Workers platform configuration. During this maintenance window, customers will be unable to make configuration changes for up to 3 minutes. This includes dashboard edits, API requests that modify configuration, and new Workers/Pages deployments.<br /><br />Affected products will continue operating normally during this time. Only management and configuration actions in the dashboard and public API will be impacted. Read-only actions, including dashboard browsing, read-only API requests, and analytics, will not be affected.<br /><br />Affected products include:<br />* Workers<br />* Pages<br />* Workers for Platforms<br />* Queues<br />* Durable Objects<br />* Workflows<br />* Tail Workers<br />* KV (namespace deletion only)<br />* Cron Triggers<br />* Workers Custom Domains<br />* Workers Subdomains </p>
