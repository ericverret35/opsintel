<p><strong>THIS IS A SCHEDULED EVENT May <var>26</var>, <var>12:00</var> - <var>13:00</var> UTC</strong></p><p><small>May <var>14</var>, <var>05:10</var> UTC</small><br /><strong>Scheduled</strong> - Cloudflare will be performing scheduled maintenance on systems supporting Rulesets configuration. During this window, Dashboard and API requests for Rulesets-based products, including WAF, Rate Limiting, Transform Rules, Redirect Rules, Cache Rules, Configuration Rules, Page Shield, and API Shield, will be intermittently unavailable, and configuration changes to rules and rulesets may not succeed. Existing rule evaluation and traffic enforcement at the Cloudflare edge will continue operating normally.</p>
